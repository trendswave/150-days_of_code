# Day 28 0f 150 days challenge
## Prime number checker
```
Prompts the user to enter a number 
Takes the number as input and checks if it is a prime number.
A prime number is a positive integer greater than 1 that has no positive divisors other than 1 and itself.
Check if the number is divisible by any integer from 2 to the square root of the number (inclusive).
 If the number is divisible by any of those integers, it is not a prime number.
 Return True if the number is prime and False otherwise.
Test your program with various examples, including prime and non-prime numbers.

```
